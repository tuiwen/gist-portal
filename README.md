# gist-portal

A client portal for Github's Gist.

## Install & Run

```
git clone https://github.com/tuiwen/gist-portal.git {your-project-folder}`
```

Then type `npm install`, then `npm test`, then `npm start`.

## Usage & Features

Visit `http:///localhost:3000` to view the docs.
